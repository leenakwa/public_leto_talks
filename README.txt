# leto talks — рейтинги учителей (демо)

Готовый статический сайт (HTML/CSS/JS без бэкенда). Всё хранится в `localStorage` браузера.
Позже легко подключить настоящую авторизацию по `@student.letovo.ru` и бэкенд для хранения данных.

## Как запустить
Откройте `index.html` двойным кликом. Работает без сервера.

## Что реализовано
- Главная страница с карточками:
  - «Топ по характеристикам» — превью из 3 учителей на каждую характеристику.
  - «По кафедрам» — превью из 3 лучших по общему рейтингу.
- Страницы списков:
  - `/top/{characteristic}` — сортировка по выбранной характеристике.
  - `/department/{кафедра}` — сортировка по OVERALL.
  - `/search?q=Фамилия` — поиск по ФИО.
- Профиль учителя:
  - Данные (ФИО, кафедра, предметы, фото-заглушка).
  - Оценка по звёздам (1–5) по каждой характеристике.
  - Общий рейтинг (среднее по характеристикам).
  - Анонимные комментарии.
- Авторизация (заглушка): кнопка «Залогиниться» просто включает режим Student (для демо).

## Добавление/удаление характеристик
В `app.js` массив `CHARACTERISTICS`. UI строится автоматически, ничего больше менять не нужно.

## Импорт учителей (200+)
В `app.js` есть функция `App.importCSV(file)` — можно добавить кнопку `<input type="file">` и вызвать её.
Формат CSV:
```
firstName,lastName,patronymic,department,subjects,photo
Иван,Петров,Алексеевич,Математика,Алгебра|Геометрия,
...
```

## Цвета
Используется синий/жёлтый из палитры Летово. Логотип школы на сайт не вставлен.
В шапке — текстовый логотип проекта «leto talks».

## Хранилище
Все данные (учителя, оценки, комментарии) лежат в `localStorage` под ключом `letotalks:*`.
Чтобы сбросить, очистите localStorage для этого домена/файла в настройках браузера.
